# Project basic rules

## 基本信息
这是一个为 flash 游戏《爆枪突击》中的“开发者”功能打造的第三方图形化编辑器与 Mod 分享平台。前端使用 Vue 3 + Vite + TypeScript，后端使用 Node.js + Express + TypeScript。

## Rules
- 每个任务完成后，使用 git commit 命令提交当前的修改。commit 的描述为你完成任务的摘要。
- 前端的 ui 设计使用 naive-ui，请优先使用相关组件。

## 设计要求
- 本项目以前端内容为主，目前尚处于页面设计阶段，不需要考虑后端/数据库相关内容。
- 整体页面设计风格参照 naive-ui 的暗色风格，请确保修改或新增的页面不突兀。
- 原游戏中的”开发者“模式需要玩家通过自主输入 xml 数据，通过游戏作者在 as3 代码中预设的接口读取。因此 xml 数据的编辑及其依赖游戏源码的设计，该工具的目的就是通过可视化编辑降低用户玩家设计门槛。因此在设计对应页面之前，务必读取对应提供的游戏源码文件，确保设计结果符合游戏逻辑。

## 文件相关说明

### 文件拆分说明
为了保证读取/操作/维护的效率，部分代码文件按功能拆分成了子文件，请在需要的时候按标注位置读取子文件。

### 项目文件结构
/project-root
┣ server # 后端架构
┣ client # 前端架构
┗ materials # 游戏源码，需要读取的辅助文件
